FFmpeg命令记录

发布 2023, Aug 12 需要1分钟读完

介绍

FFmpeg 是一款强大的音频和视频处理工具。可以说所有与音视频相关的操作都可以通过它完成。它的作者也是一个传奇。

安装

FFmpeg 官方只提供了源码，没有提供可执行文件，但是推荐了一些可以下载编译好的可执行文件的网站，具体可以看FFmpeg的官网。

命令

本文主要是记录一些我使用过的 FFmpeg 命令。并且命令的用途和每个参数的含义。

选取一个文件的音频和另一个文件的视频合并为一个文件

ffmpeg -i 1.mp4 -i 2.mp4 -c copy -map 0:v:0 -map 1:a:0 输出文件名.mp4

-i 输入文件

-c copy 复制原文件的音频和视频的编码器

-map 0:v:0 选择第一个文件的第一个视频流

-map 1:a:0 选择第二个文件的第一个音频流

剪切视频

ffmpeg  -i ./input.mp4 -c copy -ss 00:00:10 -to 00:00:15 ./output.mp4

-i 输入文件

-c copy 复制原文件的音频和视频的编码器

-ss 00:00:10 开始时间

-to 00:00:15 结束时间
