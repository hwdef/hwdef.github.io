配置linux服务器免密登录

发布 2020, Jun 11 需要1分钟读完

每次通过 ssh 登录 linux 服务器都要输密码，很麻烦。

可以通过将本机的 ssh 公钥放到 linux 服务器中，就可以免密登录。

具体操作为

  1. 先找到本机的 ssh 公钥，一般都在 .ssh 文件夹下，文件后缀为 .pub，复制文件的全部内容

  2. 登录 linux 服务器，进入 .ssh 文件夹，修改 authorized_keys 文件（如果不存在的话就创建），将上一步的 ssh 公钥复制到这个文件中

  3. 修改 authorized_keys 的权限

chmod 600 authorized_keys
  1. 如还不能登录，需根据系统修改 ssh 配置文件，开启免密登录的设置
