去除k8s节点的污点（taint）

发布 2020, Jan 02 需要1分钟读完

首先查看 node 的对应规则

kubectl describe node master |grep -E '(Roles|Taints)'

其中 master 为对应 node 的名字。

之后修改查询出来的规则

kubectl taint node master node-role.kubernetes.io/master-

同样，master 也是名字，最后的-代表去除规则

输出应该是类似 untaint 之类的。
